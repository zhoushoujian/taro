# taro-miXingFeng

```taro小程序实践```

## Useage

你需要先下载微信开发者工具,然后在project.config里填写你自己的appid才能使用  

没有微信小程序的appid可在这注册:https://mp.weixin.qq.com/wxopen/waregister?action=step1  

开发支付宝小程序只需支付宝小程序开发者工具即可,无需appid  

```shell
npm i
npm run dev:weapp
```

## Experience

微信小程序搜索```觅星峰```,开始体验!

## Functions

1. 小程序注册,登录,websocket和调用觅星峰的一些api接口

2. 这个仓库是对多平台小程序的一个实践
